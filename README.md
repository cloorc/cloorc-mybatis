# Project for Verification of Mybatis 3+

 Mybatis 3+ Example

## Frameworks

> 1. Spring Boot 1.3.5.RELEASE
> 1. Mybatis 3.4.0
> 1. Mybatis Spring 1.3.0
> 1. Mybatis Spring Boot 1.1.1
> 1. MySQL MariaDB Client 1.2.3
> 1. Slf4j 1.7.21

## Features

> 1. Mybatis Annotation with Spring Boot auto-configuration.
